# Goal
Execute the steps defined in an implementation plan to deliver a feature or fix.

# When To Use
Use this skill after a plan has been created and you are ready to write code.

# Instructions
1. Load the implementation plan.
2. Follow the detailed steps in the plan.
3. Update the execution report as you progress.

# Constraints
- Follow the plan strictly unless a divergence is justified.
- Document all divergences.

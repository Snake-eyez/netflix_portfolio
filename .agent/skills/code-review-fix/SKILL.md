# Goal
Systematically fix issues identified in a code review.

# When To Use
Use this skill when you have a list of issues from a code review that need to be addressed.

# Instructions
1. Read the code review feedback and scope.
2. For each fix: understand the issue, implement the fix, and verify.
3. Run validation commands after all fixes are applied.

# Constraints
- Address issues one by one.
- Verify each fix with tests.
